# The Oracle

An AI-powered academic assistant built with:

- Next.js
- FastAPI
- SQLite
- Ollama
- RAG Document Search

## Features

- PDF Upload
- Assignment Extraction
- Quiz Extraction
- Course Dashboard
- Due Soon Tracking
- Calendar View
- Assignment Status Tracking
- AI Study Planning (in progress)

## Future Features

- Study Planner
- OpenAI / Claude Support
- Desktop App
- Gmail Integration
- MyLearningSpace Integration
- Notifications
